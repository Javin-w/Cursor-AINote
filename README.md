# AI笔记 - 个人笔记和任务管理应用

一款现代化的个人笔记和任务待办管理产品，支持Web和移动端访问。

## ✨ 核心功能

### 📝 笔记管理
- **快捷输入**: 首页提供快速创建笔记的入口
- **智能搜索**: 支持按标题和内容搜索笔记
- **标签管理**: 为笔记添加标签便于分类和查找
- **富文本编辑**: 支持格式化文本和长文本内容
- **最近笔记**: 首页展示最近创建/更新的笔记

### ✅ 任务管理
- **任务创建**: 支持标题、描述、优先级和截止时间
- **状态管理**: 待处理、进行中、已完成三种状态
- **优先级设置**: 高、中、低三个优先级级别
- **智能筛选**: 按状态和优先级筛选任务
- **紧急提醒**: 首页显示高优先级和即将到期的任务

### 🏠 智能首页
- **快捷操作**: 直接在首页快速创建笔记
- **数据概览**: 任务统计和完成情况一目了然
- **近期动态**: 最近笔记和紧急任务聚合展示
- **响应式设计**: 完美适配桌面和移动设备

## 🛠 技术栈

### 后端
- **Node.js + Express**: 轻量级后端API服务
- **TypeScript**: 类型安全的开发体验
- **SQLite**: 嵌入式数据库，便于部署
- **数据持久化**: 自动创建和管理数据库

### 前端
- **React + TypeScript**: 现代化前端框架
- **Vite**: 快速的开发构建工具
- **Tailwind CSS**: 原子化CSS框架
- **React Router**: 单页面应用路由
- **Axios**: HTTP客户端
- **React Hot Toast**: 优雅的消息提示

## 🚀 快速开始

### 环境要求
- Node.js 18+ 
- npm 或 yarn

### 一键安装
```bash
# 运行安装脚本（推荐）
./setup.sh
```

### 开发模式
```bash
# 同时启动前后端开发服务器
npm run dev

# 创建演示数据（可选）
./demo-data.sh

# 访问应用
# 前端: http://localhost:3000
# 后端API: http://localhost:3001/api
```

### 分别启动
```bash
npm run dev:server  # 后端服务器 (http://localhost:3001)
npm run dev:client  # 前端服务器 (http://localhost:3000)
```

### 生产部署
```bash
# 方式1: 使用启动脚本（推荐）
./start.sh

# 方式2: 手动构建和启动
npm run build     # 构建前后端应用
npm start         # 启动生产服务器
```

## 🔗 API 接口

### 笔记接口
- `GET /api/notes` - 获取笔记列表
- `POST /api/notes` - 创建笔记
- `GET /api/notes/:id` - 获取单个笔记
- `PUT /api/notes/:id` - 更新笔记
- `DELETE /api/notes/:id` - 删除笔记
- `GET /api/notes/search/:query` - 搜索笔记

### 任务接口
- `GET /api/tasks` - 获取任务列表
- `POST /api/tasks` - 创建任务
- `GET /api/tasks/:id` - 获取单个任务
- `PUT /api/tasks/:id` - 更新任务
- `DELETE /api/tasks/:id` - 删除任务
- `GET /api/tasks/stats/overview` - 获取任务统计
